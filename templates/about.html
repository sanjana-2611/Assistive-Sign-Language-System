About the Sign Language Coaching & Translation System

This project is designed to make communication between hearing‑impaired individuals and the general population feel natural, intuitive, and human. It blends modern computer vision with language intelligence to move beyond literal recognition into meaningful understanding.

The problem

Sign language is expressive and powerful, but only a small percentage of hearing people can understand it. In classrooms, workplaces, hospitals, and public spaces, this gap creates silence where there should be conversation.

Existing tools often translate one sign at a time or require expensive dedicated hardware. They struggle to turn sequences of signs into sentences that capture the intent behind the gesture.

The solution

This system turns any standard camera into an intelligent sign‑language companion. Using YOLOv11‑based models trained on letters and words, it detects hand signs in real time and maps them to tokens.

A Large Language Model then rewrites these tokens into natural, grammatically correct sentences, so what appears on screen or is spoken aloud feels like a human explanation, not just a raw transcription.

Core components

  • YOLOv11 for detection

    Three custom‑trained detectors: one for alphabet letters (lettersbest.pt) and two for different word vocabularies (wordbest.pt and word2best.pt), running together on each frame.

  • LLM for language shaping

    Recognized signs become word sequences, which an LLM refines into smooth sentences, handling ordering, grammar, and context‑aware phrasing.

  • Flask web experience

    A lightweight web application hosts the full experience: landing page, explanation, and a live prediction studio with camera input, animated feedback, and sentence playback.

Impact & future work

As a prototype, this project demonstrates how vision models and language models can cooperate to support accessibility and education. With more training data, richer vocabularies, and bidirectional translation (speech‑to‑sign), it can grow into a full assistive platform.

The goal is not to replace human interpreters, but to provide a scalable bridge where none exists today—making spontaneous, everyday communication more inclusive for everyone.
